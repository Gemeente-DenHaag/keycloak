Keycloak 17

1. Change source path in below statement
2. Run command

docker run --name keycloak_test -p 8080:8080 \
      --mount type=bind,source=/Users/borai/Developer/keycloak/keycloakdhzgwpubliek,target=/opt/keycloak/themes/keycloakdhzgwpubliek \
      --mount type=bind,source=/Users/borai/Developer/keycloak/denhaagtheme,target=/opt/keycloak/themes/denhaagtheme \
      -e KEYCLOAK_ADMIN=admin -e KEYCLOAK_ADMIN_PASSWORD=change_me \
      quay.io/keycloak/keycloak:17.0.0 \
      start-dev

3. localhost:8080 for admin dashboard, locahost:8080:auth for template 


Check the readme.txt to configure the styling
